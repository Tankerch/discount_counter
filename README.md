# Discount Counter

CLI Program untuk menghitung diskon ganda pada faktur.

## Maksudnya?

Sebagai contoh, diberikan diskon ganda 5% + 2%, maka perhitungan diskonnya adalah

$$
 Harga\ Barang * (1-5\%) * (1-2\%)
$$

Tetapi, diskon ganda tersebut bisa disederhakan menjadi

$$
 Harga\ Barang * (1-6.9\%)
$$

Contoh tersebut menggunakan 2 diskon ganda, bagaimana jika ada 3, 4, atau lebih diskon? Program ini menjawab hal tersebut.

## Cara pakai

Masukan diskon ganda dipisahkan dengan koma (,). Jika diskon berbentuk desimal, gunakan titik (.) sebagai pemisah desimal

Contoh : 5% + 2%

Maka input yang dimasukan adalah 5,2
